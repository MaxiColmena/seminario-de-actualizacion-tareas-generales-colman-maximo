# 🧠 Modelos de Machine Learning en JavaScript

## 📋 Descripción

Este proyecto demuestra el uso de modelos de Machine Learning preentrenados ejecutándose directamente en el navegador web utilizando TensorFlow.js. Implementa dos funcionalidades principales: detección de lenguaje tóxico en texto y clasificación de imágenes para detectar camiones.

## ✨ Características

- **Detector de Toxicidad**: Analiza texto en tiempo real y detecta lenguaje ofensivo usando el modelo de toxicidad de TensorFlow.js
- **Clasificador de Imágenes**: Utiliza MobileNet para clasificar imágenes y detectar camiones
- **Interfaz Web**: Interfaz simple y intuitiva construida con HTML, CSS y JavaScript puro
- **Ejecución en Navegador**: No requiere instalación de software adicional, funciona directamente en el navegador
- **Resultados con Probabilidades**: Muestra las probabilidades de cada clasificación para mayor transparencia

## 🚀 Cómo Usar

1. **Clona o descarga** este repositorio
2. **Abre el archivo** `index_completo.html` en tu navegador web
3. **Espera** a que se carguen los modelos (aparecerá "Cargando modelo..." inicialmente)
4. **Usa las funcionalidades**:
   - **Texto**: Escribe en el área de texto y haz clic en "Analizar" para detectar toxicidad
   - **Imagen**: Selecciona una imagen y haz clic en "Analizar Imagen" para clasificarla

## 🛠️ Tecnologías Utilizadas

- **TensorFlow.js**: Framework para ejecutar modelos de ML en JavaScript
- **Modelo de Toxicidad**: Para análisis de texto ofensivo
- **MobileNet**: Modelo de clasificación de imágenes preentrenado
- **HTML5/CSS3**: Para la interfaz de usuario
- **JavaScript ES6+**: Lógica de la aplicación

## 📋 Requisitos

- Navegador web moderno con soporte para JavaScript (Chrome, Firefox, Safari, Edge)
- Conexión a internet para cargar los modelos desde CDN

## ⚠️ Limitaciones

- El modelo de toxicidad funciona mejor en inglés. Los resultados en español pueden no ser precisos.
- La clasificación de imágenes requiere imágenes de buena calidad para mejores resultados.
- Los modelos se cargan desde CDN, por lo que se necesita conexión a internet.
