🧠 Modelos de Machine Learning en JavaScript

Explicación

Este proyecto utiliza TensorFlow.js para ejecutar modelos de Machine Learning en el navegador. Se implementan dos modelos preentrenados: - Modelo de toxicidad: analiza texto y detecta lenguaje ofensivo. - Modelo MobileNet: clasifica imágenes y permite detectar camiones. No se entrenan modelos, solo se consumen mediante JavaScript.

Nota: El modelo de toxicidad funciona mejor en inglés, por lo que los resultados en español pueden no ser precisos.
