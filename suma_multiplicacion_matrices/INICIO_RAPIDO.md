# 🚀 INICIO RÁPIDO

## Elige tu versión

### 1️⃣ Versión Local (MÁS RÁPIDO - Sin instalación)
**Windows/Mac/Linux:**
```
1. Ve a la carpeta: local/
2. Abre el archivo: index.html
3. ¡Listo! Comienza a usar
```

---

### 2️⃣ Versión Node.js (Profesional)
**Windows:**
```bash
cd nodejs
npm install
npm start
```
Luego abre: http://localhost:3000

**Mac/Linux:**
```bash
cd nodejs
npm install
npm start
```
Luego abre: http://localhost:3000

---

### 3️⃣ Versión Python/Flask (Para Python lovers)
**Windows:**
```bash
cd python
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
python server.py
```
Luego abre: http://localhost:5000

**Mac/Linux:**
```bash
cd python
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
python server.py
```
Luego abre: http://localhost:5000

---

## ⚡ En Cualquier Versión

1. **Especifica dimensiones** → Ingresa filas y columnas
2. **Dibuja las matrices** → Click en "Dibujar Matrices"
3. **Carga los valores** → Números en cada celda
4. **Realiza operación** → Suma o Multiplicación
5. **Ve el resultado** → ¡Listo!

---

## ✅ Validaciones
- ✓ Todos los campos deben completarse
- ✓ Solo se aceptan números
- ✓ Números enteros y decimales permitidos
- ✓ Números negativos permitidos

## 🆘 Problemas?

**"Puerto ya en uso"**
→ Cambia el puerto en el archivo server.js o server.py

**"Module not found"**
→ Ejecuta: `npm install` (Node.js) o `pip install -r requirements.txt` (Python)

**"Cannot GET /"**
→ Verifica que el servidor esté corriendo
